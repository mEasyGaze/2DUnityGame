# 遊戲名稱

## 專案簡介
這是一款仍在開發中的遊戲原型，主要展示核心玩法與系統設計。

## 開發工具
- Unity
- C#

## 目前完成內容
- 一、 玩家、互動模組
- 二、 物品與背包模組
- 三、 對話系統模組
- 四、 任務系統模組
- 五、 成員與隊伍模組
- 六、 交易系統模組
- 七、 探索系統模組
- 八、 教學系統模組
- 九、 劇情系統模組
- 十、 輸入模組
- 十一、 遊戲世界模組
- 十二、 存檔系統模組
- 十三、 戰鬥系統模組
- 十四、 技能系統模組
- 十五、 智能 AI 戰鬥模組
- 十六、音樂音效模組

## 尚未完成 / 未來規劃
- 關卡、劇情擴充
- 數值調整
- 裝備系統模組
- 音效與美術強化

## 操作方式
- WASD 或 上下左右 移動
- 滑鼠操作點擊
- E互動、空白鍵拾取、P隊伍欄、B戰鬥隊伍、I背包、Q任務、ESC關閉/開啟菜單
